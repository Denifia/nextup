# Documentation

This directory contains the detailed `nextup` documentation.

## Files

- [`ai-assistant-install.md`](./ai-assistant-install.md) - agent install/integration contract with full input/output schemas; meant to be handed directly to an assistant or coding agent
- [`spec.md`](./spec.md) - behavior contract, request/response schemas, resolution rules, strategies, DST semantics, and canonical examples
- [`examples.md`](./examples.md) - practical CLI usage recipes and common request patterns
- [`agent-integration-example.md`](./agent-integration-example.md) - example walkthrough of using the agent install/integration contract in a personal assistant workflow
- [`architecture.md`](./architecture.md) - implementation structure, module responsibilities, design decisions, and testing notes

## Recommended reading order

1. `ai-assistant-install.md`
2. `spec.md`
3. `examples.md`
4. `agent-integration-example.md`
5. `architecture.md`
